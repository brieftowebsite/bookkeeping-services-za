# Bookkeeping Services ZA

Static website for Erica Woolcott's bookkeeping and small business admin support service.

## Business positioning

Calm, practical bookkeeping support for small businesses in Edenvale, Johannesburg, Gauteng and remote South Africa.

## Pages

- `index.html` — Home
- `about.html` — About Erica / business overview
- `services.html` — Bookkeeping services
- `payroll.html` — Payroll admin support
- `sars-compliance.html` — SARS and compliance admin support
- `small-business-admin.html` — Small business admin support
- `contact.html` — Netlify contact form
